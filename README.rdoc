== Welcome to [Oogway's List](http://oogwayslist.herokuapp.com/)
This is a place where you can keep track of all of those movies you wanted to watch. And who you wanted to watch them with. This is also a place that will tell you what to watch once you’ve told it how much you liked watching what you've watched and how much you think you’d like to watch what you haven’t.

== Current Features
* Create movie lists with yourself or as many members as you'd like.
* Algorithm to tell you how much each movies matches with the members.
* Hooked up to IMDb api and OMDb API http://www.omdbapi.com/ for movie information.
* Invite system to accept or deny invites to lists.
* Fully responsive layout built with Bootstrap.

== Main Technology Used
* Ruby on Rails
* AngularJS
* Twitter Bootstrap
